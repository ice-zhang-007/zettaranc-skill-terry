## 💡 这是什么？

这是一个 **GitHub Copilot 自定义 Skill**，能让 AI 完美模拟 B站主播「Z说球鞋」（Z哥/师尊/师傅）的说话风格和交易方法论。

不是冷冰冰的 AI 回答，而是像在二楼听Z哥唠嗑一样 —— 有脾气、有金句、有干货。

### ✨ 核心能力

| 能力 | 描述 |
|:---:|:---|
| 🗣️ **风格还原** | 口语化表达、毒舌幽默、生活比喻、口头禅一个不少 |
| 📈 **交易教学** | 黄线白线、活跃市值、去弱留强、数砖、B一B二……完整方法论 |
| 🔍 **知识检索** | 基于 200+ 真实直播文稿，引用Z哥原话回答问题 |
| 💬 **闲聊互动** | 人生建议、养生劝告、感情分析，"天一黑就睡觉" |

## 🚀 快速开始

### 前置条件

- [VS Code](https://code.visualstudio.com/) + [GitHub Copilot](https://marketplace.visualstudio.com/items?itemName=GitHub.copilot) 扩展
- Copilot Chat 已启用自定义 Skill 功能

### 使用方式

1. **克隆仓库**
   ```bash
   git clone https://github.com/your-username/zettaranc.git
   ```

2. **在 VS Code 中打开项目**
   ```bash
   code zettaranc
   ```

3. **在 Copilot Chat 中调用 Skill**
   
   直接在对话中提问，Copilot 会自动匹配 `z-shuoqiuxie` 技能：
   
   ```
   什么是去弱留强？
   帮我讲讲黄线白线系统
   Z哥怎么看龙队的操作？
   ```
